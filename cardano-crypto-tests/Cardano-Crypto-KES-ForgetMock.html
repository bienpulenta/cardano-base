data ForgetMockKES k #

A wrapper for a KES implementation that adds logging functionality, for the purpose of verifying that invocations of genKeyKES and forgetSignKeyKES pair up properly in a given host application.

The wrapped KES behaves exactly like its unwrapped payload, except that invocations of genKeyKES, updateKES and forgetSignKeyKES are logged as GenericEvent ForgetMockEvent values. (We use GenericEvent in order to use the generic MonadSodium instance of LogT; otherwise we would have to provide a boilerplate instance here).
