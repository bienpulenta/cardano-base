noExceptionsThrown :: Applicative m => m Property #

Return a property that always succeeds in some monad (typically IO). This is useful to express that we are only interested in whether the side effects of the preceding actions caused any exceptions or not - if they did, then the test will fail because of it, but if they did not, then noExceptionsThrown will be reached, and the test will succeed.
